## 📁 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/lrdplopes/nodejs-challenge1

    #Acessar o repositório
    $ cd nodejs-challenge1

    #Instalar suas dependências
    $yarn install

    #Iniciar o projeto
    $yarn start

```
